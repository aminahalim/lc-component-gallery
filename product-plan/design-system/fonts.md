# Typography Configuration

## Google Fonts Import

Add to your HTML `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&display=swap" rel="stylesheet">
```

## Font Family

**Montserrat** is the sole typeface for all UI text.

```css
font-family: 'Montserrat', sans-serif;
```

## Type Scale

| Size | Weight | Usage |
|------|--------|-------|
| 32px | Bold (700) | Page titles |
| 24px | Bold (700) | Modal titles |
| 20px | Bold (700) | Section headings |
| 16px | SemiBold (600) | Card titles |
| 15px | Bold (700) | Widget titles, nav labels |
| 14px | Medium (500) | Body text, button labels, table cells |
| 13px | Regular (400) | Secondary body text, descriptions |
| 12px | Medium (500) | Sidebar labels, metadata, filter tabs |
| 11px | SemiBold (600) | Sidebar icon labels, compact nav |
| 10px | Bold (700) uppercase | Table headers, section labels, filter group labels |
